# Ocean UI

Ocean Professional themed Angular component library.

- Primary: `#2563EB`
- Secondary: `#F59E0B`
- Surface: `#ffffff`
- Text: `#111827`

## Usage

1. Install and build the library in this workspace.
2. Import `OceanUiModule.forRoot()` in your root module or provide theme via `OCEAN_THEME`.

```ts
import { OceanUiModule } from '@ocean/ui';

@NgModule({
  imports: [BrowserModule, OceanUiModule.forRoot()]
})
export class AppModule {}
```

Explore components in the "Components" section.
